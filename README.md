# This is a Next.js project with 
-   Styled-components
-   Type script


## Getting Started


```bash
git clone https://github.com/micrometre/nextjs-ts
cd nexjs-ts
```

```bash
npm install
# or
yarn 
```



```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
